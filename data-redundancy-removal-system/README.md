# Data Redundancy Removal System

A Streamlit-based web application that detects and removes duplicate records from CSV files while storing clean data in an SQLite database.

## Features

- CSV File Upload
- Duplicate Detection
- Remove Duplicate Records
- Download Cleaned CSV
- SQLite Database Storage
- Manual Data Entry
- Email Validation
- Phone Number Validation
- Duplicate Email Prevention
- Search Records
- Update Records
- Delete Records
- Dashboard with Statistics
- Export Database to CSV

## Technologies Used

- Python
- Streamlit
- Pandas
- SQLite
- ReportLab

## Project Structure

```
Data-Redundancy-Removal-System/
│── app.py

```

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
